#  Terraform AWS Infrastructure Project

This project automates the creation of **AWS infrastructure** using **Terraform** — including **VPC, EC2, IAM, and S3**

## ⚙️ How to Use

1. **Initialize Terraform**
   ```bash
   terraform init
2. **Preview Resources**
   terraform plan
3. **Apply configuration**
   terraform apply -auto-approve
4. **View outputs**
   terraform output
